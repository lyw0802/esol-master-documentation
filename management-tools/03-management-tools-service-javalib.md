# java lib(jar) 관리

---

## 1. java lib(jar) 관리란
### 1.1. 정의

>DB Service에서 java method 호출 가능한 custom java lib(jar)을 관리 하는 기능,  
>사용자가 직접 jar 파일을 등록 할 수 없다,  
>ESOL 운영주체만 jar 파일을 등록 할 수 있으며 등록된 jar 파일만 관리 


## 2. 사용법
### 2.1. 등록

🎈 __Menu > 관리도구 > java lib(jar) > 등록__
